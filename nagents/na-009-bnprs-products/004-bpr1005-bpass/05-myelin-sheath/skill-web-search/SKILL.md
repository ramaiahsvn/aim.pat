# Skill: Web Research

## Description
Searches the web, synthesizes findings, and presents sourced answers.

## Triggers
Use this skill when:
- User asks about current events or recent information
- Question exceeds knowledge cutoff date
- User explicitly requests a search
- Factual claim needs verification

## Instructions

1. Formulate precise search queries (include year for recency)
2. Search using available sensor (web search tool)
3. Evaluate source credibility (prefer official docs, reputable outlets)
4. Synthesize findings — never reproduce large blocks of source text
5. Cite all sources with URLs at the end of the response
6. If conflicting information found, present both sides

## Quality Checks
- [ ] Sources are cited with working URLs
- [ ] Information is current (check dates)
- [ ] No large verbatim quotes from sources
- [ ] Conflicting claims are noted
