# 01-resources

Everything the agent can read or call.

| Sub-folder   | What goes here |
|-------------|----------------|
| `connectors/` | MCP server configs, API endpoint definitions |
| `files/`      | Reference docs, lookup tables, static data the agent reads |
| `secrets/`    | `secrets.yaml` (git-ignored) — credentials and tokens |

Copy `secrets/secrets.example.yaml` → `secrets/secrets.yaml` and fill in values.
